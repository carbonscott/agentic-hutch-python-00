## Accessing hutch-python

Log into hutch-python node, we need to run hutch-python as a control operator
(e.g. rixopr)

```
ssh psdev
ssh rixopr@rix-daq
```

hutch-python is accessed by running `rix3`, it will log you into a ipython
session directly.
